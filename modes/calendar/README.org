** Org integration

   When =evil-collection-calendar-want-org-bindings= is non-nil, the key named by
   =org-calendar-to-agenda-key= (default =c=) is bound to =org-calendar-goto-agenda=,
   and =org-calendar-insert-diary-entry-key= (default =i=) to =org-agenda-diary-entry=
   when =org-agenda-diary-file= is set.

** Customization

   | Variable                                     | Default | Purpose                                     |
   |----------------------------------------------+---------+---------------------------------------------|
   | =evil-collection-calendar-want-org-bindings= | =nil=   | Install =org-calendar-*= bindings on entry. |
