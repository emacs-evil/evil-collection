* newsticker

  Evil bindings for [[https://www.gnu.org/software/emacs/manual/html_node/newsticker/index.html][newsticker]], the built-in RSS / Atom reader.

** Key bindings

*** newsticker-mode-map (plainview)

    | Command                                     | Original | Evil |
    |---------------------------------------------+----------+------|
    | =newsticker-previous-feed=                  | =F=      | =gk= |
    | =newsticker-next-feed=                      | =f=      | =gj= |
    | =newsticker-mark-item-at-point-as-read=     | =o=      | =r=  |
    | =newsticker-mark-item-at-point-as-immortal= | =i=      | =i=  |
    | =newsticker-show-old-items=                 | =s O=    | =o=  |
    | =newsticker-hide-old-items=                 | =h O=    | =O=  |
    | =newsticker-buffer-force-update=            | =U=      | =gr= |
    | =newsticker-get-all-news=                   | =G=      | =gR= |
    | =newsticker-close-buffer=                   | =q=      | =q=  |

** Notes

   - In the treeview list buffer, =C-m= / =C-j= are bound to
     =newsticker-treeview-show-item= via text-property keymap, which has higher
     precedence than the major-mode map. Use =M-j= / =M-k= for next / previous
     item navigation.
